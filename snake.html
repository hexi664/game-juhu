Snake - Classic Arcade Game

Control the snake, eat food to grow longer, and avoid hitting the walls or your own body. A classic arcade-style game that's easy to learn but challenging to master.

Game Controls:

- Arrow Keys (↑ ↓ ← →): Control snake movement direction

- Space Bar: Pause/Resume game

- P Key: Pause/Resume game

- R Key: Restart game
